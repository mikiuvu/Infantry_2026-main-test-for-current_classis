# 全向轮底盘功率控制实现对比分析

## 一、功率估算模型

### 1.1 理论公式

两种实现都基于相同的电机功率模型：

$$P(I, \omega) = K_0 + K_1 \cdot I + K_2 \cdot \omega + K_3 \cdot I \cdot \omega + K_4 \cdot I^2 + K_5 \cdot \omega^2$$

| 参数 | 物理意义 | 单位 |
|------|----------|------|
| $K_0$ | 静态功耗（空载损耗） | W |
| $K_1 \cdot I$ | 电流线性项 | W |
| $K_2 \cdot \omega$ | 速度线性项（反电动势相关） | W |
| $K_3 \cdot I \cdot \omega$ | 机械功率项（力矩×角速度） | W |
| $K_4 \cdot I^2$ | 铜损（电阻热，欧姆定律） | W |
| $K_5 \cdot \omega^2$ | 铁损/风阻（与速度平方成正比） | W |

---

### 1.2 文档方案实现

```c
/**
 * @brief 文档方案 - 功率预测函数
 * @param motor 电机实例指针
 * @param motor_type 电机类型 (M3508/GM6020)
 * 
 * 特点：
 * 1. 使用 real_current（电机反馈的实际电流）
 * 2. 根据电机类型使用不同的K参数
 * 3. GM6020对速度取绝对值
 */
void DJIMotorPowerPredict(DJIMotorInstance *motor, Motor_Type_e motor_type)
{   
    if(motor_type == M3508)
    {
        // M3508电流转换: 反馈值范围 [-16384, +16384] 对应 [-20A, +20A]
        float current = motor->measure.real_current * 20.0f / 16384.0f;  // 单位: A
        
        // 速度转换: degree/s -> rad/s
        float speed = motor->measure.speed_aps * DEGREE_2_RAD;  // 单位: rad/s
        
        // 使用M3508专用参数计算功率
        motor->predicted_power = K0_3508 
                               + K1_3508 * current 
                               + K2_3508 * speed 
                               + K3_3508 * current * speed 
                               + K4_3508 * POW2(current) 
                               + K5_3508 * POW2(speed);
    }
    
    if(motor_type == GM6020)
    {
        // GM6020电流转换: 反馈值范围 [-16384, +16384] 对应 [-3A, +3A]
        float current = motor->measure.real_current * 3.0f / 16384.0f;  // 单位: A
        
        // 速度转换并取绝对值（功率与转向无关）
        float speed = motor->measure.speed_aps * DEGREE_2_RAD;
        speed = speed > 0 ? speed : -speed;  // 取绝对值
        
        // 使用GM6020专用参数计算功率
        motor->predicted_power = K0_6020 
                               + K1_6020 * current 
                               + K2_6020 * speed 
                               + K3_6020 * current * speed 
                               + K4_6020 * POW2(current) 
                               + K5_6020 * POW2(speed);
    }
}
```

## 三、电流衰减系数计算

### 3.1 数学原理

当总功率超过限制时，需要找到一个衰减系数 $\lambda$，使得：

$$\sum_{i=1}^{4} P_i(\lambda \cdot I_i, \omega_i) = P_{max}$$

将功率模型展开：

$$\sum_{i=1}^{4} \left[ K_0 + K_1 \cdot \lambda I_i + K_2 \cdot \omega_i + K_3 \cdot \lambda I_i \cdot \omega_i + K_4 \cdot (\lambda I_i)^2 + K_5 \cdot \omega_i^2 \right] = P_{max}$$

整理为关于 $\lambda$ 的二次方程：

$$a \cdot \lambda^2 + b \cdot \lambda + c = 0$$

其中：
- $a = \sum K_4 \cdot I_i^2$
- $b = \sum I_i \cdot (K_1 + K_3 \cdot \omega_i)$
- $c = \sum (K_0 + K_2 \cdot \omega_i + K_5 \cdot \omega_i^2) - P_{max}$

---

### 3.2 文档方案实现

```c
/**
 * @brief 文档方案 - 二次方程求解衰减系数
 * 
 * 目标: 找到 λ ∈ (0, 1)，使总功率 = P_max
 */

if(total_power > max_power_motor)
{
    float a = 0, b = 0, c = 0;
    
    // 遍历所有底盘电机，累加二次方程系数
    for(uint8_t i = 0; i < POWER_LIMIT_3508_CNT; i++)
    {
        if(dji_motor_powerLimit_3508[i] != NULL)
        {
            // 获取当前电机的速度和电流
            float speed = dji_motor_powerLimit_3508[i]->measure.speed_aps * DEGREE_2_RAD;
            float current = dji_motor_powerLimit_3508[i]->measure.real_current * 20.0f / 16384.0f;
            
            // 累加二次项系数 a = Σ K4 * I²
            a += K4_3508 * POW2(current);
            
            // 累加一次项系数 b = Σ I * (K1 + K3*ω)
            b += K1_3508 * current + K3_3508 * current * speed;
            
            // 累加常数项 c = Σ (K0 + K2*ω + K5*ω²)
            c += K0_3508 + K2_3508 * speed + K5_3508 * POW2(speed);
        }
    }
    
    // c 减去目标功率，使方程右边为0
    c -= max_power_motor;
    
    // 计算判别式 Δ = b² - 4ac
    float delta = b * b - 4 * a * c;
    
    if(delta < 0)
    {
        // 无实数解: 无论怎么降低电流都无法满足功率限制
        // 通常发生在速度很高时（铁损占主导）
        limit_coef = 0;  // 直接停止输出
    }
    else
    {
        // 求根公式: λ = (-b ± √Δ) / 2a
        float x1 = (-b + sqrtf(delta)) / (2 * a);  // 解1
        float x2 = (-b - sqrtf(delta)) / (2 * a);  // 解2
        
        // 选择在 (0, 1) 区间内的解
        if((x1 > 0 && x1 < 1) || (x2 > 0 && x2 < 1))
        {
            // 取有效解（优先取较大的，减少性能损失）
            limit_coef = (x1 > 0 && x1 < 1) ? x1 : x2;
        }
        else
        {
            // 两个解都不在 (0, 1) 区间
            limit_coef = 0;
        }
    }
}
else
{
    // 未超功率，不限制
    limit_coef = 1;
}

// 应用衰减系数到PID输出
set *= limit_coef;
```

## 四、全向轮底盘特殊考虑

### 4.1 运动学特性

全向轮（X型布局）的逆运动学：

```c
/**
 * @brief 全向轮逆运动学
 * 
 * 底盘速度 (vx, vy, wz) -> 四轮速度 (v_lf, v_rf, v_lb, v_rb)
 * 
 * 特点: 四个轮子的负载分布会随运动方向变化
 */

// 各轮目标速度计算 (chassis.c)
vt_lf = (-chassis_vx - chassis_vy) / Sqrt(2) + wz * CENTER2;
vt_rf = (chassis_vx - chassis_vy) / Sqrt(2) - wz * CENTER1;
vt_lb = (-chassis_vx + chassis_vy) / Sqrt(2) - wz * CENTER3;
vt_rb = (chassis_vx + chassis_vy) / Sqrt(2) + wz * CENTER4;
```

### 4.2 功率分配不均匀问题

```
运动方向对功率分配的影响:

    纯前进 (vx > 0, vy = 0):
    ┌─────────────────┐
    │  LF: 中等负载    │  RF: 中等负载
    │       ↑         │
    │      前进方向    │
    │                 │
    │  LB: 中等负载    │  RB: 中等负载
    └─────────────────┘
    
    斜向运动 (vx > 0, vy > 0):
    ┌─────────────────┐
    │  LF: 高负载  ↗  │  RF: 低负载
    │                 │
    │                 │
    │  LB: 低负载     │  RB: 高负载
    └─────────────────┘

    自旋 (wz > 0):
    ┌─────────────────┐
    │  LF ──→         │  ←── RF
    │       ↻         │
    │      旋转       │
    │  LB ──→         │  ←── RB
    └─────────────────┘
    四轮负载相近
```

### 4.3 对功率限制的影响
