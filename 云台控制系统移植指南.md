从完整的 RoboMaster Infantry 工程中提取**云台控制核心功能**,移植到新的空工程中。

- ✅ 遥控器(DR16)控制
- ✅ 视觉通信接口

- ✅ IMU 姿态解算(BMI088 + IST8310)

删除的功能:
- ❌ 底盘控制(M3508)
- ❌ 发射机构(摩擦轮、拨弹)
- ❌ 裁判系统
- ❌ UI 绘制
- ❌ 其他应用模块

## 🔧 第一步: STM32CubeMX 配置

### 1.1 时钟配置
```
外部晶振: 12 MHz
SYSCLK: 168 MHz
APB1: 42 MHz (CAN, USART2/3)
APB2: 84 MHz (USART1)
```

### 1.2 外设配置

#### CAN1 - 电机通信
```
模式: Master
波特率: 1 Mbps
引脚:
  - PD0: CAN1_RX
  - PD1: CAN1_TX
中断: 使能 CAN1_RX0_IRQn (优先级 5)
```

#### USART1 - 视觉通信
```
模式: Asynchronous
波特率: 115200
数据位: 8
停止位: 1
校验: None
引脚:
  - PA9:  USART1_TX
  - PA10: USART1_RX
DMA:
  - USART1_RX: DMA2 Stream 2 (优先级 High)
  - USART1_TX: DMA2 Stream 7 (优先级 Medium)
中断: 使能 USART1_IRQn + DMA 中断
```

#### USART3 - 遥控器(DR16)
```
模式: Asynchronous
波特率: 100000
数据位: 8 + Even Parity (实际 9 位)
停止位: 1
校验: Even
引脚:
  - PD8: USART3_TX
  - PD9: USART3_RX (或 PA3,根据硬件)
DMA:
  - USART3_RX: DMA1 Stream 1 (优先级 Very High)
中断: 使能 USART3_IRQn + DMA 中断
```

#### SPI1 - IMU(BMI088)
```
模式: Full-Duplex Master
时钟: 10.5 MHz
数据大小: 8 Bits
极性: Low
相位: 1 Edge
NSS: Software
引脚:
  - PA5: SPI1_SCK
  - PA6: SPI1_MISO
  - PA7: SPI1_MOSI
  - PA4: 加速度计 CS (GPIO Output)
  - PB0: 陀螺仪 CS (GPIO Output)
中断: 可选 EXTI 中断(数据就绪引脚)
```

#### I2C3 - 磁力计(IST8310)
```
模式: I2C
速度: 400 kHz
引脚:
  - PA8: I2C3_SCL
  - PC9: I2C3_SDA
```

#### TIM5 - DWT 高精度延时
```
模式: Internal Clock
预分频: 168-1 (1 MHz)
计数周期: 0xFFFFFFFF
```

### 1.3 FreeRTOS 配置
```
内核版本: CMSIS_V2
任务调度器: Preemptive
CPU 频率: 168 MHz
Tick 频率: 1000 Hz

任务配置:
  - defaultTask: 删除或改为监控任务
  - 最小堆栈: 128 Words (512 Bytes)

添加任务:
  1. INSTask:     优先级 Normal,  栈 1024 Words
  2. MotorTask:   优先级 High,    栈 512 Words
  3. RobotTask:   优先级 Normal,  栈 512 Words
```

### 1.4 中断优先级规划
```
FreeRTOS 最大中断优先级: 5

中断优先级分配:
  - USART3 (遥控器):    抢占 4, 子优先级 0  (高于 FreeRTOS)
  - CAN1_RX0:           抢占 5, 子优先级 0
  - USART1:             抢占 5, 子优先级 1
  - DMA 中断:           抢占 5, 子优先级 2
  - TIM 中断:           抢占 6, 子优先级 0
```

**修改 `robot_cmd.c` - 简化版:**
```c
#include "robot_cmd.h"
#include "robot_def.h"
#include "remote_control.h"
#include "ins_task.h"
#include "master_process.h"
#include "message_center.h"

#define YAW_ALIGN_ANGLE (YAW_CHASSIS_ALIGN_ECD * ECD_ANGLE_COEF_DJI)
#define PTICH_HORIZON_ANGLE (PITCH_HORIZON_ECD * ECD_ANGLE_COEF_DJI)

static RC_ctrl_t *rc_data;
static Vision_Recv_s *vision_recv_data;
static Vision_Send_s vision_send_data;

static Publisher_t *gimbal_cmd_pub;
static Subscriber_t *gimbal_feed_sub;
static Gimbal_Ctrl_Cmd_s gimbal_cmd_send;
static Gimbal_Upload_Data_s gimbal_fetch_data;

static Robot_State_e robot_state;

void RobotCMDInit()
{
    rc_data = RemoteControlInit(&huart3);       // 遥控器初始化
    vision_recv_data = VisionInit(&huart1);     // 视觉初始化
    
    gimbal_cmd_pub = PubRegister("gimbal_cmd", sizeof(Gimbal_Ctrl_Cmd_s));
    gimbal_feed_sub = SubRegister("gimbal_feed", sizeof(Gimbal_Upload_Data_s));
    
    gimbal_cmd_send.pitch = PTICH_HORIZON_ANGLE;  // 初始水平位置
    gimbal_cmd_send.yaw = 0;
    gimbal_cmd_send.gimbal_mode = GIMBAL_ZERO_FORCE;
    robot_state = ROBOT_READY;
}

/**
 * @brief 遥控器控制模式
 */
static void RemoteControlSet()
{
    // 摇杆灵敏度
    #define YAW_RC_SENS 0.003f    // yaw 灵敏度
    #define PITCH_RC_SENS 0.002f  // pitch 灵敏度
    
    // 云台模式设置
    if (switch_is_down(rc_data[TEMP].rc.switch_right)) {
        gimbal_cmd_send.gimbal_mode = GIMBAL_GYRO_MODE;  // 陀螺仪模式
    }
    
    // 遥控器手动控制
    if (switch_is_down(rc_data[TEMP].rc.switch_left)) {
        // 右摇杆控制 yaw 和 pitch
        gimbal_cmd_send.yaw   -= rc_data[TEMP].rc.rocker_r_ * YAW_RC_SENS;
        gimbal_cmd_send.pitch += rc_data[TEMP].rc.rocker_r1 * PITCH_RC_SENS;
    }
    
    // 视觉自瞄模式
    else if (switch_is_mid(rc_data[TEMP].rc.switch_left)) {
        gimbal_cmd_send.yaw = vision_recv_data->aimYaw;
        gimbal_cmd_send.pitch = vision_recv_data->aimPitch;
    }
    
    // Pitch 角度限幅
    if (gimbal_cmd_send.pitch >= PITCH_MAX_ANGLE) {
        gimbal_cmd_send.pitch = PITCH_MAX_ANGLE;
    }
    else if (gimbal_cmd_send.pitch <= PITCH_MIN_ANGLE) {
        gimbal_cmd_send.pitch = PITCH_MIN_ANGLE;
    }
}

/**
 * @brief 安全急停检查
 */
static void EmergencyHandler()
{
    // 遥控器离线或拨轮过高
    if (!RemoteControlIsOnline() || rc_data[TEMP].rc.dial >= 640) {
        robot_state = ROBOT_STOP;
        gimbal_cmd_send.gimbal_mode = GIMBAL_ZERO_FORCE;  // 失能
        
        // 急停时保持当前角度,避免恢复时突变
        gimbal_cmd_send.yaw = gimbal_fetch_data.gimbal_imu_data.YawTotalAngle;
        gimbal_cmd_send.pitch = gimbal_fetch_data.gimbal_imu_data.Pitch;
    }
    else {
        robot_state = ROBOT_READY;
    }
}

/**
 * @brief 机器人核心控制任务, 200Hz 运行
 */
void RobotCMDTask()
{
    // 获取云台反馈数据
    SubGetMessage(gimbal_feed_sub, &gimbal_fetch_data);
    
    // 安全检查
    EmergencyHandler();
    
    // 遥控器控制
    if (robot_state == ROBOT_READY) {
        RemoteControlSet();
    }
    
    // 发布云台控制指令
    PubPushMessage(gimbal_cmd_pub, &gimbal_cmd_send);
    
    // 发送视觉数据(可选)
    // vision_send_data.yaw = gimbal_fetch_data.gimbal_imu_data.Yaw;
    // vision_send_data.pitch = gimbal_fetch_data.gimbal_imu_data.Pitch;
    // VisionSend(&vision_send_data);
}

void GimbalInit()
{
    gimba_IMU_data = INS_Init();
    
    // Yaw 电机配置
    Motor_Init_Config_s yaw_config = {

            .can_handle = &hcan1,

        },
        .controller_setting_init_config = {
            .angle_feedback_source = OTHER_FEED,
            .speed_feedback_source = OTHER_FEED,
            .outer_loop_type = ANGLE_LOOP,
            .close_loop_type = ANGLE_LOOP | SPEED_LOOP,
        },
        .motor_type = GM6020
    };
    
    // Pitch 电机配置(类似)
    Motor_Init_Config_s pitch_config = {
        // ... (保持你当前的配置)
    };
    
    yaw_motor = DJIMotorInit(&yaw_config);
    pitch_motor = DJIMotorInit(&pitch_config);
    
    gimbal_pub = PubRegister("gimbal_feed", sizeof(Gimbal_Upload_Data_s));
    gimbal_sub = SubRegister("gimbal_cmd", sizeof(Gimbal_Ctrl_Cmd_s));
}

void GimbalTask()
{
    SubGetMessage(gimbal_sub, &gimbal_cmd_recv);
    
    switch (gimbal_cmd_recv.gimbal_mode) {
        case GIMBAL_ZERO_FORCE:
            DJIMotorStop(yaw_motor);
            DJIMotorStop(pitch_motor);
            break;
            
        case GIMBAL_GYRO_MODE:
            DJIMotorEnable(yaw_motor);
            DJIMotorEnable(pitch_motor);
            DJIMotorSetRef(yaw_motor, gimbal_cmd_recv.yaw);
            DJIMotorSetRef(pitch_motor, gimbal_cmd_recv.pitch);
            break;
    }
    
    // 反馈数据
    gimbal_feedback_data.gimbal_imu_data = *gimba_IMU_data;
    gimbal_feedback_data.yaw_motor_single_round_angle = 
        yaw_motor->measure.angle_single_round;
    
    PubPushMessage(gimbal_pub, &gimbal_feedback_data);
}

osThreadId_t MotorTaskHandle;
osThreadId_t RobotTaskHandle;

// INS 姿态解算任务 (1000Hz)
void INS_Task_Entry(void *argument)
{
    for(;;)
    {
        INS_Task();
        osDelay(1);  // 1ms
    }
}

// 电机控制任务 (500Hz)
void Motor_Task_Entry(void *argument)
{
    for(;;)
    {
        DJIMotorControl();
        osDelay(2);  // 2ms
    }
}

// 机器人主任务 (200Hz)
void Robot_Task_Entry(void *argument)
{
    RobotInit();  // 机器人初始化
    
    for(;;)
    {
        RobotTask();
        osDelay(5);  // 5ms
    }
}

void MX_FREERTOS_Init(void)
{
    // 创建任务
    const osThreadAttr_t INS_attr = {
        .name = "INSTask",
        .stack_size = 1024 * 4,
        .priority = osPriorityNormal,
    };
    INSTaskHandle = osThreadNew(INS_Task_Entry, NULL, &INS_attr);
    
    const osThreadAttr_t Motor_attr = {
        .name = "MotorTask",
        .stack_size = 512 * 4,
        .priority = osPriorityHigh,
    };
    MotorTaskHandle = osThreadNew(Motor_Task_Entry, NULL, &Motor_attr);
    
    const osThreadAttr_t Robot_attr = {
        .name = "RobotTask",
        .stack_size = 512 * 4,
        .priority = osPriorityNormal,
    };
    RobotTaskHandle = osThreadNew(Robot_Task_Entry, NULL, &Robot_attr);
}
```

## 🐛 常见问题与调试

### 问题 1: 编译错误 "未定义引用"
**原因:** Makefile 中缺少源文件
**解决:** 检查 `C_SOURCES` 是否包含所有 `.c` 文件

### 问题 2: 硬件错误(HardFault)
**原因:** 
- 空指针访问
- 栈溢出
- 中断优先级冲突

**调试方法:**
```c
// 在 HardFault_Handler 中添加
void HardFault_Handler(void)
{
    printf("HardFault at: 0x%08X\n", __get_PSP());
    while(1);
}
```

### 问题 3: 遥控器离线
**检查:**
1. USART3 波特率是否为 100000
2. 是否开启奇偶校验(Even Parity)
3. DMA 是否正确配置
4. 守护进程 `reload_count` 是否合适

### 问题 4: IMU 数据异常
**检查:**
1. SPI 时钟频率(建议 10 MHz)
2. CS 引脚是否正确
3. 加速度计和陀螺仪是否都初始化成功
4. 磁力计 I2C 通信是否正常

### 问题 5: 电机不转
**检查:**
1. CAN 总线是否有 120Ω 终端电阻
2. 电机 ID 是否正确(Yaw=7, Pitch=5)
3. 电机是否使能(`DJIMotorEnable`)
4. PID 参数是否合理

## ⚠️ 注意事项

1. **不要在中断中调用 FreeRTOS API** (除了 `FromISR` 版本)
2. **不要在中断中使用 `printf`** (会导致死锁)
3. **确保所有指针初始化后再使用** (避免空指针)
4. **CAN 总线必须有终端电阻** (120Ω)
5. **遥控器必须先对频** (接收器绿灯常亮)
6. **电机 ID 不能冲突** (同一总线)
7. **IMU 必须水平放置校准** (上电时静止)
