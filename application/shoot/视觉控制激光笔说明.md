# 视觉控制激光笔说明

## 📡 通信协议概述

下位机支持三种视觉通信协议,通过修改 `robot_def.h` 中的宏定义选择:

1. **VCP协议** (USB虚拟串口) - `#define VISION_USE_VCP`
2. **UART协议** (串口Seasky协议) - `#define VISION_USE_UART`
3. **SerialPort协议** (HUST上位机) - `#define VISION_USE_SERIALPORT`

---

## 🎯 激光笔控制逻辑

### 触发条件

激光笔在以下**所有条件同时满足**时打开:

✅ `shoot_mode == SHOOT_ON` (发射系统开启)  
✅ `friction_mode == FRICTION_ON` (摩擦轮开启)  
✅ `load_mode == LOAD_BURSTFIRE` (连发模式)

---

## 📤 上位机发送格式

### 1️⃣ VCP/UART 协议

**数据结构** (`Vision_Recv_s`):
```c
typedef struct {
    uint16_t head;      // 包头标识(0xA5)
    uint8_t fire;       // 🔥 开火标志: 1=开火, 0=停火
    uint8_t tracking;   // 跟踪标志: 1=跟踪中, 0=未跟踪
    float aimYaw;       // 目标Yaw角度(度)
    float aimPitch;     // 目标Pitch角度(度)
} Vision_Recv_s;       // 总共12字节
```

**控制激光笔的字段**:
- `fire = 1` → 激光笔**打开** ✅
- `fire = 0` → 激光笔**关闭** ❌

**Python示例代码**:
```python
import struct

# 打开激光笔
data = struct.pack('<HBBff', 
    0xA5,      # head (小端序uint16)
    1,         # fire = 1 (开火)
    1,         # tracking = 1 (跟踪中)
    5.5,       # aimYaw (float)
    -3.2       # aimPitch (float)
)
serial_port.write(data)

# 关闭激光笔
data = struct.pack('<HBBff', 0xA5, 0, 0, 0.0, 0.0)
serial_port.write(data)
```

### 2️⃣ SerialPort 协议 (HUST)

**数据结构** (`SerialPort_Recv_s`):
```c
typedef struct {
    uint8_t startflag;      // '!' (0x21)
    uint8_t flag;           // 0x05
    int16_t pitch;          // Pitch角度*100 (大端序)
    int32_t yaw;            // Yaw角度*100 (大端序)
    uint8_t shootStatus;    // 🔥 射击状态: 非0=开火, 0=停火
    uint32_t time_stamp;    // 时间戳 (大端序)
    uint8_t state;          // 状态
    uint8_t num;            // 目标编号
    uint8_t crc;            // CRC8校验
} SerialPort_Recv_s;       // 总共16字节
```

**控制激光笔的字段**:
- `shootStatus != 0` → 激光笔**打开** ✅
- `shootStatus == 0` → 激光笔**关闭** ❌

**Python示例代码**:
```python
import struct

def crc8_calc(data):
    # CRC8计算函数 (需要根据实际协议实现)
    crc = 0
    for byte in data:
        crc ^= byte
        for _ in range(8):
            if crc & 0x80:
                crc = (crc << 1) ^ 0x31
            else:
                crc <<= 1
            crc &= 0xFF
    return crc

# 打开激光笔
pitch = int(-3.2 * 100)  # 转换为整数
yaw = int(5.5 * 100)
data_without_crc = struct.pack('>BBhiBIBB',
    0x21,      # startflag '!'
    0x05,      # flag
    pitch,     # pitch (大端序int16)
    yaw,       # yaw (大端序int32)
    1,         # shootStatus = 1 (开火)
    12345678,  # time_stamp (大端序uint32)
    1,         # state
    0          # num
)
crc = crc8_calc(data_without_crc)
data = data_without_crc + bytes([crc])
serial_port.write(data)

# 关闭激光笔
data_without_crc = struct.pack('>BBhiBIBB',
    0x21, 0x05, 0, 0, 0, 12345678, 0, 0)
crc = crc8_calc(data_without_crc)
data = data_without_crc + bytes([crc])
serial_port.write(data)
```

## 🎮 控制模式

### 遥控器模式

当**左侧开关在中位**时,进入视觉模式:

| 条件 | 激光笔状态 |
|------|-----------|
| 视觉发送 `fire=1` / `shootStatus≠0` | 🟢 打开 |
| 视觉发送 `fire=0` / `shootStatus=0` | 🔴 关闭 |

**摩擦轮自动开启**,无需额外控制。

---

### 键鼠模式

当**按下鼠标右键**时,开启自瞄模式:

| 条件 | 激光笔状态 |
|------|-----------|
| 鼠标右键 + 视觉 `fire=1` | 🟢 打开 |
| 鼠标右键 + 视觉 `fire=0` | 🔴 关闭 |
| 鼠标左键按下 | 🟢 打开(手动) |
| 鼠标左键松开 + 视觉不开火 | 🔴 关闭 |

**摩擦轮需要按 F 键控制**。

1. 连接USB或UART1
2. 遥控器左侧开关拨到**中位**(视觉模式)
3. 遥控器拨轮向上拨(开启摩擦轮)

   停火: A5 00 00 00 00 00 00 00 00 00 00 00

## 🐛 故障排查

| 问题 | 检查项 |
|------|--------|
| 视觉有数据但激光笔不亮 | ✓ 左侧开关在中位<br>✓ 摩擦轮已开启<br>✓ `fire`/`shootStatus`字段=1 |
| 激光笔一直亮 | ✓ 检查视觉发送频率<br>✓ 确认`fire`字段正确设置为0 |
| 协议不匹配 | ✓ `robot_def.h`宏定义<br>✓ 上位机协议选择 |
| 串口无数据 | ✓ 串口配置(波特率等)<br>✓ 线缆连接<br>✓ 协议头校验 |

---

## 📝 注意事项

1. **协议选择**: 确保下位机和上位机使用相同的协议
2. **字节序**: SerialPort协议使用大端序,VCP/UART使用小端序
3. **数据类型**: VCP/UART使用float,SerialPort使用int(需要×100)
4. **CRC校验**: SerialPort协议需要正确计算CRC8
5. **发送频率**: 建议50-100Hz,过高可能导致通信拥堵
6. **包头验证**: VCP/UART使用0xA5,SerialPort使用0x21
