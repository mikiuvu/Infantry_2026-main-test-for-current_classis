## 1. 整体架构

### 1.1 控制框图

```
                          ┌──────────────────────────────────────────────────────────┐
                          │                    力控前馈通路                           │
                          │                                                          │
目标速度 ──┬──> 速度误差 ──> 鲁棒力控制器 ──> 目标力Fx,Fy ──> 摩擦自适应限幅 ──┐      │
           │              (滑模+DOB)                                          │      │
           │                                                                  ▼      │
           │                                              力分配 ──> 力-电流映射 ──> 电流前馈 ──┐
           │                                              (逆运动学)  (自适应kt)              │
           │                                                                                  │
           └──────────────────────────────────────────────────────────────────────────────────┤
                                                                                              │
                          ┌───────────────────────────────────────────────────────────────────┤
                          │                    速度环兜底通路                                  │
                          │                                                                   │
目标速度 ──> 逆运动学 ──> 各轮目标速度 ──> 电机速度环(软) ──> 速度环输出 ──────────────────────┤
                                                                                              │
                                                                                              ▼
                                                                              电流指令 = 速度环输出 + 电流前馈
                                                                                              │
                                                                                              ▼
                                                                                         电机电流环
                                                                                              │
                                                                                              ▼
                                                                                           电机输出
```

### 1.2 设计原则

| 通路 | 作用 | 特点 |
|------|------|------|
| **力控前馈** | 快速响应、主动补偿 | 基于模型预测，响应快 |
| **速度环兜底** | 消除稳态误差、防漂移 | 低增益，不干预动态 |

### 1.3 与现有代码的关系

当前 `chassis.c` 已有的架构：
- `DJIMotorInit` 配置了 `SPEED_LOOP | CURRENT_LOOP` 双闭环
- `current_feedforward_ptr` 指向电流前馈变量
- `GlobalObserverCalculate()` 计算前馈电流

**升级方向**：将 `GlobalObserverCalculate()` 替换为更完善的力控计算。

## 2. 基础力速混控实现

### 2.1 物理模型

#### 2.1.1 底盘动力学
```
牛顿第二定律:
    m * a_x = F_x + d_x        (X方向)
    m * a_y = F_y + d_y        (Y方向)
    J * α_z = M_z + d_z        (旋转)

其中:
    m = 底盘质量 (kg)
    J = 转动惯量 (kg*m²)
    F_x, F_y, M_z = 控制力/力矩
    d_x, d_y, d_z = 扰动 (摩擦、坡度、碰撞等)
```

#### 2.1.2 电机力矩模型
```
M3508电机:
    τ_motor = k_t * I                    (电流-力矩)
    F_wheel = τ_motor / R_wheel / η      (力矩-轮子力)
    
合并:
    F_wheel = k_total * I
    
其中:
    k_t = 0.3 N*m/A                      (M3508力矩常数)
    R_wheel = 0.076 m                    (轮子半径)
    η = 19.2                             (减速比)
    k_total = k_t / R_wheel / η ≈ 0.00206 N/A
```

### 2.2 力分配 (逆运动学)

#### 2.2.1 X型麦轮/全向轮力分配
```c
// 底盘力 -> 各轮力 (N)
// 麦轮安装角度45°，X型布局
F_lf = (-Fx - Fy) / (2 * sqrt(2)) + Mz / (4 * L);
F_rf = ( Fx - Fy) / (2 * sqrt(2)) - Mz / (4 * L);
F_lb = (-Fx + Fy) / (2 * sqrt(2)) - Mz / (4 * L);
F_rb = ( Fx + Fy) / (2 * sqrt(2)) + Mz / (4 * L);

// 其中 L = 轮子到中心的距离 (m)
```

#### 2.2.2 力-电流映射
```c
// 力 -> 电流 (A -> 电调单位需要乘系数)
I_lf = F_lf / k_total;
I_rf = F_rf / k_total;
I_lb = F_lb / k_total;
I_rb = F_rb / k_total;

// M3508电调: 1A ≈ 819.2 电调单位 (16384/20A)
current_ff_lf = I_lf * 819.2f;  // 或直接用 F_lf / kt_estimate
```

### 2.3 代码实现

```c
/* ===================== 车辆参数 ===================== */
#define CHASSIS_MASS        15.0f       // 整车质量 (kg)
#define CHASSIS_INERTIA_Z   0.8f        // Z轴转动惯量 (kg*m²)
#define WHEEL_RADIUS        0.076f      // 轮子半径 (m)
#define WHEEL_BASE_L        0.2f        // 轮距/2 (m)

/* ===================== 力矩常数 ===================== */
// M3508: k_t = 0.3 N*m/A, 减速比19.2, 轮径0.076m
// k_total = 0.3 / 19.2 / 0.076 = 0.00206 N/A
// 考虑效率损失，初始值取90%: 0.00185 N/A
#define KT_INITIAL          0.00185f    // 初始力矩常数 (N/A)

/* ===================== 基础力控计算 ===================== */
typedef struct {
    float Fx;           // X方向目标力 (N)
    float Fy;           // Y方向目标力 (N)
    float Mz;           // Z轴目标力矩 (N*m)
} ChassisForceCmd_t;

static ChassisForceCmd_t force_cmd = {0};

/**
 * @brief 速度误差 -> 目标力 (简单PD控制，后续替换为鲁棒控制)
 */
static void BasicForceCalculate(float vx_ref, float vy_ref, float wz_ref,
                                 float vx_act, float vy_act, float wz_act)
{
    // 速度误差
    float vx_err = vx_ref - vx_act;
    float vy_err = vy_ref - vy_act;
    float wz_err = wz_ref - wz_act;
    
    // PD控制 -> 期望加速度 -> 期望力
    // F = m * a = m * (Kp * e + Kd * e_dot)
    // 简化: 假设 e_dot ≈ -Kp * e (一阶近似)
    static float Kp_v = 20.0f;   // 速度环比例增益
    static float Kp_w = 5.0f;    // 角速度环比例增益
    
    force_cmd.Fx = CHASSIS_MASS * Kp_v * vx_err;
    force_cmd.Fy = CHASSIS_MASS * Kp_v * vy_err;
    force_cmd.Mz = CHASSIS_INERTIA_Z * Kp_w * wz_err;
}

/**
 * @brief 力分配与电流映射
 * @param kt 当前力矩常数估计值 (N/A)
 */
static void ForceAllocation(float kt)
{
    float Fx = force_cmd.Fx;
    float Fy = force_cmd.Fy;
    float Mz = force_cmd.Mz;
    
    // 力分配 (X型全向轮)
    float F_lf = (-Fx - Fy) / (2.0f * Sqrt(2)) + Mz / (4.0f * WHEEL_BASE_L);
    float F_rf = ( Fx - Fy) / (2.0f * Sqrt(2)) - Mz / (4.0f * WHEEL_BASE_L);
    float F_lb = (-Fx + Fy) / (2.0f * Sqrt(2)) - Mz / (4.0f * WHEEL_BASE_L);
    float F_rb = ( Fx + Fy) / (2.0f * Sqrt(2)) + Mz / (4.0f * WHEEL_BASE_L);
    
    // 力 -> 电流前馈 (电调单位)
    // 注意: kt 的单位是 N/A，需要转换到电调单位
    // M3508: 1A = 819.2 电调单位
    float kt_esc = kt * 819.2f;  // N/电调单位
    
    current_ff_lf = F_lf / kt_esc;
    current_ff_rf = F_rf / kt_esc;
    current_ff_lb = F_lb / kt_esc;
    current_ff_rb = F_rb / kt_esc;
}
```

## 3. 在线自适应参数估计

### 3.1 原理

**问题**：力矩常数 `k_t` 会因为电机温度、负载、电池电压等因素变化。

**解决**：通过比较 **模型预测加速度** 与 **IMU实际加速度**，实时修正 `k_t`。

#### 3.1.1 自适应律推导

```
模型: a_expect = (k_t * I) / m
实际: a_actual = IMU测量

误差: e = a_actual - a_expect = a_actual - (k̂_t * I) / m

MIT自适应律:
    dk̂_t/dt = γ * e * ∂e/∂k̂_t = γ * e * (-I/m)
    
简化 (离散化):
    k̂_t(k+1) = k̂_t(k) + γ * (a_actual - a_expect) * I / (1 + I²)
    
归一化因子 1/(1+I²) 防止大电流时过度调整
```

### 3.2 代码实现

```c
/* ===================== 在线自适应参数估计 ===================== */
typedef struct {
    float kt_estimate;      // 估计的力矩常数 (N/A)
    float kt_min;           // 最小允许值
    float kt_max;           // 最大允许值
    float gamma;            // 自适应增益
} AdaptiveParam_t;

static AdaptiveParam_t adaptive = {
    .kt_estimate = KT_INITIAL,  // 初始值
    .kt_min = 0.0010f,          // 最小 (约55%标称)
    .kt_max = 0.0025f,          // 最大 (约135%标称)
    .gamma = 0.00001f,          // 自适应增益 (需调试)
};

/**
 * @brief 在线自适应更新力矩常数
 * @param I_x X方向等效电流 (电调单位)
 * @param I_y Y方向等效电流 (电调单位)
 * @param ax_actual IMU实测X加速度 (m/s²)
 * @param ay_actual IMU实测Y加速度 (m/s²)
 * @param dt 时间步长 (s)
 */
static void AdaptiveUpdate(float I_x, float I_y, 
                            float ax_actual, float ay_actual, float dt)
{
    // 电调单位 -> 安培
    float I_x_amp = I_x / 819.2f;
    float I_y_amp = I_y / 819.2f;
    float I_norm = sqrtf(I_x_amp * I_x_amp + I_y_amp * I_y_amp);
    
    // 只在有足够激励时更新 (避免低电流时噪声主导)
    if (I_norm < 2.0f) return;  // 小于2A不更新
    
    // 模型预测加速度
    float ax_expect = adaptive.kt_estimate * I_x_amp / CHASSIS_MASS;
    float ay_expect = adaptive.kt_estimate * I_y_amp / CHASSIS_MASS;
    
    // 预测误差
    float error_x = ax_actual - ax_expect;
    float error_y = ay_actual - ay_expect;
    
    // 归一化自适应律 (MIT Rule)
    float norm_factor = 1.0f / (1.0f + I_norm * I_norm);
    float delta_kt = adaptive.gamma * (error_x * I_x_amp + error_y * I_y_amp) 
                     * norm_factor * dt;
    
    // 更新估计值并限幅
    adaptive.kt_estimate += delta_kt;
    adaptive.kt_estimate = fmaxf(adaptive.kt_min, 
                                  fminf(adaptive.kt_max, adaptive.kt_estimate));
}

/**
 * @brief 计算等效电流 (用于自适应)
 *        将四轮电流通过正运动学合成为底盘XY方向等效电流
 */
static void GetEquivalentCurrent(float *I_x, float *I_y)
{
    // 正运动学: 各轮电流 -> XY方向等效电流
    // 与力分配相反
    *I_x = (-current_ff_lf + current_ff_rf - current_ff_lb + current_ff_rb) 
           / (2.0f * Sqrt(2));
    *I_y = (-current_ff_lf - current_ff_rf + current_ff_lb + current_ff_rb) 
           / (2.0f * Sqrt(2));
}
```

### 3.3 调试要点

| 参数 | 作用 | 调试方法 |
|------|------|----------|
| `gamma` | 收敛速度 | 从0.000001开始，逐渐增大直到kt稳定收敛 |
| `kt_min/max` | 防止发散 | 根据标定结果设置±40%范围 |
| 电流阈值 | 激励条件 | 太低会受噪声影响，太高则更新少 |

## 4. 鲁棒控制设计

### 4.1 滑模控制 (SMC)

#### 4.1.1 原理

**目标**：设计控制律使系统状态收敛到滑模面 $s=0$，且在滑模面上保持。

```
滑模面定义:
    s = ė + λ*e = (v_ref - v) + λ*(v_ref - v)的积分
    
简化为速度误差:
    s = v̇_err + λ*v_err
    
控制律:
    F = m * (λ*v̇_err + η*sat(s)) + d̂
    
其中:
    λ = 滑模面斜率 (越大收敛越快)
    η = 趋近律增益
    sat(s) = 饱和函数 (消除抖振)
    d̂ = 扰动估计
```

#### 4.1.2 代码实现

```c
/* ===================== 滑模控制器 ===================== */
typedef struct {
    float lambda;       // 滑模面斜率
    float eta;          // 趋近律增益
    float phi;          // 边界层厚度 (消除抖振)
    float disturbance;  // 估计的扰动 (由DOB提供)
} SlidingModeCtrl_t;

static SlidingModeCtrl_t smc_vx = {
    .lambda = 2.0f,     // 滑模面斜率
    .eta = 50.0f,       // 趋近律增益
    .phi = 0.3f,        // 边界层厚度 (m/s)
    .disturbance = 0.0f
};
static SlidingModeCtrl_t smc_vy = {
    .lambda = 2.0f,
    .eta = 50.0f,
    .phi = 0.3f,
    .disturbance = 0.0f
};

/**
 * @brief 饱和函数 (边界层法消除抖振)
 */
static float Saturation(float s, float phi)
{
    if (fabsf(s) < phi) {
        return s / phi;  // 边界层内线性
    } else {
        return (s > 0) ? 1.0f : -1.0f;  // 边界层外饱和
    }
}

/**
 * @brief 滑模控制器计算
 * @param smc 控制器实例
 * @param v_err 速度误差 (m/s)
 * @param a_err 加速度误差近似 (可用 -λ*v_err)
 * @return 控制力 (N)
 */
static float SlidingModeControl(SlidingModeCtrl_t *smc, float v_err, float a_err)
{
    // 滑模面
    float s = a_err + smc->lambda * v_err;
    
    // 控制律
    float u = CHASSIS_MASS * (smc->lambda * a_err + smc->eta * Saturation(s, smc->phi));
    
    // 加入扰动补偿
    u += smc->disturbance;
    
    return u;
}
```

### 4.2 扰动观测器 (DOB)

#### 4.2.1 原理

**目标**：在线估计系统未建模扰动（摩擦、坡度、外力等）。

```
系统模型:
    m * v̇ = u + d        (d是扰动)
    
观测器设计:
    ż = -L*z - L*(L*m*v + u)
    d̂ = z + L*m*v
    
其中:
    z = 观测器内部状态
    L = 观测器增益 (带宽)
    d̂ = 扰动估计值
```

#### 4.2.2 代码实现

```c
/* ===================== 扰动观测器 ===================== */
typedef struct {
    float z;            // 观测器状态
    float d_hat;        // 估计扰动 (N)
    float L;            // 观测器增益 (带宽)
} DisturbanceObserver_t;

static DisturbanceObserver_t dob_x = {.z = 0, .d_hat = 0, .L = 50.0f};
static DisturbanceObserver_t dob_y = {.z = 0, .d_hat = 0, .L = 50.0f};

/**
 * @brief 扰动观测器更新
 * @param dob 观测器实例
 * @param v 当前速度 (m/s)
 * @param u 控制输入力 (N) - 上一周期的
 * @param dt 时间步长 (s)
 * @return 估计扰动 (N)
 */
static float DOBUpdate(DisturbanceObserver_t *dob, float v, float u, float dt)
{
    float Lmv = dob->L * CHASSIS_MASS * v;
    
    // 观测器动态
    float z_dot = -dob->L * dob->z - dob->L * (Lmv + u);
    dob->z += z_dot * dt;
    
    // 扰动估计
    dob->d_hat = dob->z + Lmv;
    
    // 限幅防止发散 (最大扰动约为0.5g)
    dob->d_hat = fmaxf(-CHASSIS_MASS * 5.0f, 
                        fminf(CHASSIS_MASS * 5.0f, dob->d_hat));
    
    return dob->d_hat;
}
```

### 4.3 滑模 + DOB 整合

```c
/**
 * @brief 鲁棒力控制计算 (滑模 + 扰动观测)
 * @note 替代 BasicForceCalculate
 */
static void RobustForceCalculate(float vx_ref, float vy_ref,
                                  float vx_act, float vy_act,
                                  float dt)
{
    // 速度误差
    float vx_err = vx_ref - vx_act;
    float vy_err = vy_ref - vy_act;
    
    // 加速度误差近似 (一阶近似)
    static float vx_err_last = 0, vy_err_last = 0;
    float ax_err = (vx_err - vx_err_last) / dt;
    float ay_err = (vy_err - vy_err_last) / dt;
    vx_err_last = vx_err;
    vy_err_last = vy_err;
    
    // 滑模控制 (加速度误差可简化为 -lambda * v_err)
    float Fx = SlidingModeControl(&smc_vx, vx_err, -smc_vx.lambda * vx_err);
    float Fy = SlidingModeControl(&smc_vy, vy_err, -smc_vy.lambda * vy_err);
    
    // 更新扰动观测器 (使用上一周期的力)
    float dx = DOBUpdate(&dob_x, vx_act, force_cmd.Fx, dt);
    float dy = DOBUpdate(&dob_y, vy_act, force_cmd.Fy, dt);
    
    // 将扰动补偿传给滑模控制器
    smc_vx.disturbance = -dx;  // 负号: 补偿要抵消扰动
    smc_vy.disturbance = -dy;
    
    // 更新力指令
    force_cmd.Fx = Fx;
    force_cmd.Fy = Fy;
    // Mz 可以用类似方法或简单PD
    force_cmd.Mz = CHASSIS_INERTIA_Z * 5.0f * (0 - Chassis_INS_data->Gyro[2]); // 简化
}
```

## 5. 摩擦系数在线估计

### 5.1 原理

**目标**：估计当前轮地摩擦系数，用于限制最大输出力，避免打滑。

```
理想情况: F_ground = F_motor  →  a = F_motor / m
打滑时:   F_ground < F_motor  →  a < F_motor / m

摩擦效率 = (m * a_actual) / F_motor_cmd

当不打滑时，效率应接近1.0
当地面滑时，效率会下降
```

### 5.2 代码实现

```c
/* ===================== 摩擦系数在线估计 ===================== */
typedef struct {
    float mu_estimate;      // 当前估计摩擦效率 (0~1)
    float mu_nominal;       // 标称值 (干燥地面)
    float mu_min;           // 最小值 (湿滑地面)
    float filter_alpha;     // 低通滤波系数
    uint32_t sample_count;  // 有效样本计数
} FrictionEstimator_t;

static FrictionEstimator_t friction = {
    .mu_estimate = 0.85f,   // 初始假设
    .mu_nominal = 0.85f,
    .mu_min = 0.3f,
    .filter_alpha = 0.02f,  // 慢速滤波
    .sample_count = 0,
};

/**
 * @brief 摩擦系数在线估计
 * @param Fx_cmd X方向指令力 (N)
 * @param Fy_cmd Y方向指令力 (N)  
 * @param ax_actual 实际X加速度 (m/s²)
 * @param ay_actual 实际Y加速度 (m/s²)
 * @param max_slip_ratio 当前最大滑移率 (0~1)
 */
static void FrictionEstimatorUpdate(float Fx_cmd, float Fy_cmd,
                                     float ax_actual, float ay_actual,
                                     float max_slip_ratio)
{
    float F_cmd_norm = sqrtf(Fx_cmd * Fx_cmd + Fy_cmd * Fy_cmd);
    float a_actual_norm = sqrtf(ax_actual * ax_actual + ay_actual * ay_actual);
    
    // ===== 有效性检查 =====
    // 1. 指令力足够大
    if (F_cmd_norm < 15.0f) return;
    
    // 2. 不在严重打滑状态
    if (max_slip_ratio > 0.25f) return;
    
    // 3. 方向一致性检查
    if (F_cmd_norm > 0.01f && a_actual_norm > 0.01f) {
        float cos_angle = (Fx_cmd * ax_actual + Fy_cmd * ay_actual) 
                          / (F_cmd_norm * a_actual_norm);
        if (cos_angle < 0.7f) return;  // 夹角太大，数据不可信
    }
    
    // ===== 摩擦效率计算 =====
    float mu_measured = (CHASSIS_MASS * a_actual_norm) / F_cmd_norm;
    mu_measured = fmaxf(friction.mu_min, fminf(1.0f, mu_measured));
    
    // 低通滤波更新
    friction.mu_estimate = friction.mu_estimate * (1.0f - friction.filter_alpha)
                         + mu_measured * friction.filter_alpha;
    
    friction.sample_count++;
}

/**
 * @brief 获取当前最大可用地面力
 * @return 最大力 (N)
 */
static float GetMaxGroundForce(void)
{
    // 单轮法向力 ≈ mg/4
    float normal_force_per_wheel = CHASSIS_MASS * 9.81f / 4.0f;
    
    // 最大总摩擦力 = μ * N * 4 * 安全系数
    float safety_factor = 0.75f;  // 留25%裕量
    return friction.mu_estimate * normal_force_per_wheel * 4.0f * safety_factor;
}

/**
 * @brief 摩擦自适应力限幅
 */
static void FrictionAdaptiveLimiter(float *Fx, float *Fy)
{
    float F_max = GetMaxGroundForce();
    float F_cmd = sqrtf((*Fx) * (*Fx) + (*Fy) * (*Fy));
    
    if (F_cmd > F_max && F_cmd > 0.01f) {
        float scale = F_max / F_cmd;
        *Fx *= scale;
        *Fy *= scale;
    }
}
```

```c
static void ForceControlCalculate(float dt)
{
#if defined(CHASSIS_BOARD) || defined(CHASSIS_ONLY)
    // ========== 1. 获取状态 (单位转换: mm/s -> m/s) ==========
    float vx_ref = chassis_cmd_recv.vx / 1000.0f;
    float vy_ref = chassis_cmd_recv.vy / 1000.0f;
    float wz_ref = chassis_cmd_recv.wz * DEGREE_2_RAD;  // 假设wz输入是deg/s
    
    float vx_act = chassis_feedback_data.real_vx / 1000.0f;
    float vy_act = chassis_feedback_data.real_vy / 1000.0f;
    float wz_act = Chassis_INS_data->Gyro[2];  // rad/s
    
    float ax_act = Chassis_INS_data->MotionAccel_b[0];  // m/s² (已去重力)
    float ay_act = Chassis_INS_data->MotionAccel_b[1];
    
    // ========== 2. 鲁棒力控制 (滑模 + DOB) ==========
    RobustForceCalculate(vx_ref, vy_ref, vx_act, vy_act, dt);
    
    // 简化的Mz控制 (可扩展为单独的滑模)
    force_cmd.Mz = CHASSIS_INERTIA_Z * 5.0f * (wz_ref - wz_act);
    
    // ========== 3. 摩擦自适应估计与限幅 ==========
    float max_slip = 0;
    for (int i = 0; i < 4; i++) {
        if (wheel_slip[i].deviation > max_slip)
            max_slip = wheel_slip[i].deviation;
    }
    max_slip /= 100.0f;  // 百分比 -> 小数
    
    FrictionEstimatorUpdate(force_cmd.Fx, force_cmd.Fy, ax_act, ay_act, max_slip);
    FrictionAdaptiveLimiter(&force_cmd.Fx, &force_cmd.Fy);
    
    // ========== 4. 力分配与电流映射 ==========
    ForceAllocation(adaptive.kt_estimate);
    
    // ========== 5. 在线自适应更新 kt ==========
    float I_x, I_y;
    GetEquivalentCurrent(&I_x, &I_y);
    AdaptiveUpdate(I_x, I_y, ax_act, ay_act, dt);
    
#else
    // 无IMU模式: 清零前馈
    current_ff_lf = current_ff_rf = current_ff_lb = current_ff_rb = 0;
#endif
}
```

### 7.1 调试顺序

```
Step 1: 基础力控验证
    ├── 关闭自适应、滑模、DOB
    ├── 只用简单PD力控 + 力分配
    ├── 验证: 给定力 → IMU加速度 是否匹配
    └── 调整: kt_estimate 初始值

Step 2: 自适应调试
    ├── 开启自适应
    ├── VOFA观察 kt_estimate 收敛曲线
    ├── 验证: kt 应在标称值附近稳定
    └── 调整: gamma (收敛速度)

Step 3: DOB调试
    ├── 开启扰动观测器
    ├── 手动推车，观察 d_hat 响应
    ├── 验证: d_hat 能跟踪手推力
    └── 调整: L (观测器带宽)

Step 4: 滑模调试
    ├── 开启滑模控制
    ├── 测试速度跟踪性能
    ├── 验证: 无抖振、收敛快
    └── 调整: lambda, eta, phi

Step 5: 摩擦估计调试
    ├── 在不同地面测试
    ├── VOFA观察 mu_estimate
    ├── 验证: 干燥地面≈0.8, 湿滑≈0.4
    └── 调整: filter_alpha

Step 6: 综合测试
    ├── 全速加速测试
    ├── 急停测试
    ├── 打滑恢复测试
    └── 不同场地适应性测试
```

## 附录: 关键公式速查

### A.1 力分配 (X型麦轮)

$$
\begin{bmatrix} F_{lf} \\ F_{rf} \\ F_{lb} \\ F_{rb} \end{bmatrix} = 
\begin{bmatrix} 
-\frac{1}{2\sqrt{2}} & -\frac{1}{2\sqrt{2}} & \frac{1}{4L} \\
\frac{1}{2\sqrt{2}} & -\frac{1}{2\sqrt{2}} & -\frac{1}{4L} \\
-\frac{1}{2\sqrt{2}} & \frac{1}{2\sqrt{2}} & -\frac{1}{4L} \\
\frac{1}{2\sqrt{2}} & \frac{1}{2\sqrt{2}} & \frac{1}{4L}
\end{bmatrix}
\begin{bmatrix} F_x \\ F_y \\ M_z \end{bmatrix}
$$

### A.2 自适应律

$$
\hat{k}_t(k+1) = \hat{k}_t(k) + \gamma \cdot \frac{(a_{actual} - a_{expect}) \cdot I}{1 + I^2}
$$

### A.3 滑模控制律

$$
F = m \cdot (\lambda \dot{e} + \eta \cdot sat(s)) + \hat{d}
$$

$$
sat(s) = \begin{cases} s/\phi & |s| < \phi \\ sign(s) & |s| \geq \phi \end{cases}
$$

### A.4 扰动观测器

$$
\dot{z} = -L \cdot z - L \cdot (L \cdot m \cdot v + u)
$$

$$
\hat{d} = z + L \cdot m \cdot v
$$
